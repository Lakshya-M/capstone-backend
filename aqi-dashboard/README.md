# AQI Sensor Heatmap — Digital Twin

React app for the AQI heatmap dashboard. The three sensor positions on the floor plan are the black dots (Node 1: top-left, Node 2: bottom-left, Node 3: near wardrobe).

## Run from command line

```bash
cd aqi-dashboard
npm install
npm run dev
```

Then open **http://localhost:5173** in your browser.

## Scripts

- **`npm run dev`** — Start development server (with hot reload)
- **`npm run build`** — Build for production (output in `dist/`)
- **`npm run preview`** — Preview the production build locally
